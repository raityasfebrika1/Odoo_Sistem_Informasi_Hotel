# Odoo Sistem Informasi Hotel ARA

Project UAS Sistem Enterprise.

## Deskripsi Proyek

Sistem ini dirancang untuk mengelola operasional hotel menggunakan framework Odoo 17, mencakup modul-modul kustom untuk kebutuhan perhotelan.

## Cara Menjalankan (Local)

1. Clone repository ini.
2. Pastikan PostgreSQL menyala.
3. Jalankan command:
   `python odoo-bin -c odoo17.conf`
